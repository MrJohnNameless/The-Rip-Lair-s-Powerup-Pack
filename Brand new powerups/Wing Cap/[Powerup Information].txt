This Powerup has various cheats you can do!
These are:

* needawingcap
* overtherainbow
* sm63pilled

The powerup also has some configs you can apply. Simple open the file "wingcap.lua" in the "powerups" folder, using something like notepad++ or visual studio code.

canSlowFall - Allows the player to glide similar to using a raccoon leaf
temporary - If set to true, the powerup will last the duration set below, play music and disappear eventually.
duration - How long (in seconds) should the wing cap last for? If set to 0 or lower, it will play the music indefinitely but never disappear
music - The music file to play when the powerup is active
fligthDelay - How many frames should the player have to recover before taking off again?
invincibleWhenFalling - Should the player have some invulnerability frames when falling really fast?
